# Backend

Backend на FastAPI отвечает за API, доменную модель, импорт данных, расчет
рейтингов, auth и права доступа.

## Основные модули

- Игроки.
- Турниры.
- Соревнования.
- Команды.
- Матчи и сеты.
- Рейтинги и история рейтингов.
- Импорт внешних данных.
- Фоновые задачи.
- Пользователи, роли и области доступа.

## Принципы

- HTTP-запросы не выполняют долгие расчеты.
- Импорт и пересчет рейтингов запускаются как фоновые задачи.
- Исходные данные импорта сохраняются для отладки и воспроизводимости.
- Публичное API должно быть оптимизировано под чтение.
