# Расчет рейтинга

Расчет рейтинга выполняется на backend и запускается как фоновая задача.

## Требования

- Расчет должен быть воспроизводимым.
- История изменения рейтинга должна сохраняться.
- Должна быть возможность пересчитать рейтинг после исправления импортированных
  данных или алгоритма.
- Алгоритм расчета должен быть покрыт тестами.

## Данные

Для расчета используются:

- игроки;
- команды;
- матчи;
- сеты;
- соревнования;
- турниры;
- предыдущие значения рейтинга и история.
