# Аутентификация и права

Аутентификация реализуется внутри FastAPI-приложения.

## Auth flow

- Пользователь регистрируется по email и password.
- Backend создает пользователя со статусом неподтвержденного email.
- Backend отправляет код или ссылку подтверждения на email.
- После подтверждения пользователь может авторизоваться.
- Login возвращает access token и refresh token.

## Права

Права должны поддерживать глобальные роли и роли в рамках конкретной области.

Примеры ролей:

- Глобальный администратор.
- Администратор лиги.
- Менеджер турнира.
- Модератор контента.
- Обычный пользователь.

## Области доступа

Область доступа может соответствовать лиге, турниру, клубу или другой сущности,
если это понадобится в доменной модели.
