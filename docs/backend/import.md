# Импорт данных

Данные берутся из внешних источников через парсинг или API.

## Принципы импорта

- Сохраняем исходный payload.
- Нормализуем данные в доменные сущности.
- Повторный импорт должен обновлять существующие данные, а не создавать дубликаты.
- Ошибки импорта должны быть видны в админке.
- Импорт запускается в фоновой задаче.

## Источники

Первым целевым источником считается Kickertool и совместимые с ним данные из
прошлой версии проекта.
