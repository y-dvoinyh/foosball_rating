# Административная панель

Основная админка реализуется во frontend-приложении на Quasar и работает через
admin endpoints в FastAPI.

## Возможности

- Управление пользователями.
- Управление ролями и областями доступа.
- Запуск импорта данных.
- Просмотр статуса фоновых задач.
- Запуск пересчета рейтингов.
- Просмотр ошибок импорта и расчета.

## Принципы

- Админка должна учитывать реальные workflows проекта.
- Доступ к разделам зависит от ролей и областей пользователя.
- Долгие операции запускаются через фоновые задачи.
