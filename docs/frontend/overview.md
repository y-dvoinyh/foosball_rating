# Frontend

Frontend строится на Vue 3 + Quasar + TypeScript.

## Зоны ответственности

- Публичные страницы рейтингов.
- Профили игроков.
- Страницы турниров и соревнований.
- Графики и история рейтинга.
- Авторизация и регистрация.
- Административная панель.

## Принципы

- Публичные страницы должны быть быстрыми и удобными для просмотра.
- Админка должна быть рабочим инструментом, а не декоративной страницей.
- Таблицы, фильтры и формы строятся на компонентах Quasar.
